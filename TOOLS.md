# TOOLS.md - 本地笔记

Skill 定义工具是怎么运作的。这个文件是_你_的专属信息——你的环境特有的东西。

## 放什么在这里

比如：
- 摄像头名称和位置
- SSH 主机和别名
- TTS 偏好音色
- 音箱/房间名称
- 设备昵称
- 任何环境相关的东西

## 示例

```markdown
### 摄像头

- living-room → 主区域，180° 广角
- front-door → 入口，移动触发

### SSH

- home-server → 192.168.1.100, user: admin

### TTS

- 偏好音色："Nova"（温暖，略带英式）
- 默认音箱：厨房 HomePod
```

## 为什么分开

Skill 是共享的。你的环境是你的。分开存放意味着更新 skill 不会丢失笔记，分享 skill 不会泄露你的基础设施。
